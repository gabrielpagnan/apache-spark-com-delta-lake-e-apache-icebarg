# Apache Spark com Delta Lake e Apache Iceberg

Este repositorio foi criado para o desenvolvimento do trabalho sobre a integracao do **Apache Spark** com as tecnologias **Delta Lake** e **Apache Iceberg**, explorando conceitos de processamento distribuido, tabelas transacionais e armazenamento de dados em ambientes analiticos.

## Objetivo

O objetivo deste trabalho e apresentar e analisar como o Apache Spark pode ser utilizado em conjunto com Delta Lake e Apache Iceberg para:

- processar grandes volumes de dados;
- realizar leituras e escritas eficientes;
- garantir versionamento e confiabilidade nas tabelas;
- comparar recursos e caracteristicas das duas tecnologias.

## Tecnologias abordadas

- Apache Spark
- Delta Lake
- Apache Iceberg
- Data Lake
- Processamento distribuido de dados
